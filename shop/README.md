# Online Shop

## Description  
This project is a responsive **Home Page** and **Login page** and **Panel page** design for an online electronics store called **TechSound**.  
It showcases categories such as **earphones, headphones, and accessories**, built with **HTML, CSS, and JavaScript**.  
The purpose of this project is to practice **frontend design**, page structuring, and simple UI interactions.

---

## Features  
- Responsive layout using **Flexbox** and **CSS Grid**  
- **Navigation bar** with hover effects  
- **Hero banner** and promotional section  
- **Product cards** displaying example items  
- **Footer** with "About Us" and contact info  
- **JavaScript interactivity** for basic animations or button effects 
- Separate page for users and admin 

---

## Technologies Used  
- HTML5  
- CSS3  
- JavaScript

---

## Learning Goals  
- Strengthen understanding of **responsive layouts** (Grid/Flexbox)  
- Practice clean UI design principles  
- Learn to structure frontend projects for scalability  

---

## How to Run

1. Clone the repository or download the project.
2. Open the `Home/home.html` and `Panel admin/panel.html` and `Login/login.html` file in your browser.
