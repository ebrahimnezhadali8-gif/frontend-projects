# Learning Programming Site

## Description  
This project is a responsive design for an online learning programming , built with **HTML, CSS, and BOOTSTRAP**.  
The purpose of this project is to practice **frontend design**, page structuring, and simple UI interactions.

---

## Features  
- **Navigation bar** with hover effects  
- **Hero banner** and promotional section  
- **cards** displaying example items  
- **Footer** with "About Us" and contact info

---

## Technologies Used  
- HTML5  
- CSS3  
- BOOTSTRAP
